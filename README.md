# steam-torrents

[Install script](https://github.com/shishkevichd/steam-torrents/raw/master/steamPirate.user.js)

A Tampermonkey script that add links to torrents for Steam app page

![Screenshot](./images/screenshot.png)

## Supported sites

- RuTracker
- Steam Repacks
- Online Fix
- Skidrow
- FitGirl
- SteamRIP
- Dodi
- Gload
